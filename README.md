# USCC Pattern Recoloring Tool

A browser-based tool for recoloring PNG image sequences using the U.S. Chamber of Commerce color system.

## Setup

1. Add your source PNGs to each pattern folder under `patterns/`
2. Rename them to `00000.png` through `00120.png` (5-digit zero-padded)
3. Enable GitHub Pages (Settings → Pages → Deploy from main branch)
4. Your tool is live!

## Folder Structure

```
patterns/
├── dotted/          ← 00000.png - 00120.png
├── kaleidoscope/    ← 00000.png - 00120.png
├── linear/          ← 00000.png - 00120.png
├── pointed/         ← 00000.png - 00120.png
├── stripe/          ← 00000.png - 00120.png
└── wedge/           ← 00000.png - 00120.png
```
